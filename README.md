# xfgotool
```
xiefu的go通过工具 xf.gotool  。
补充一些其它工具未完善的方法函数

```

### 依赖的工具库：
```
https://github.com/sourcegraph/conc  优化异常捕捉，线程池

https://github.com/emirpasic/gods    数据结构相关的工具，比如：Sets, Lists, Stacks, Maps, Trees, Queues


github.com/bytedance/sonic  高性能json序列化与反序列化库

github.com/gogf/gf/v2/    含各种服务工具，建议优先使用readme，比如： github.com/gogf/gf/v2/os/glog ， github.com/gogf/gf/v2/util/gconv

github.com/shopspring/decimal    处理小数

github.com/duke-git/lancet/v2/strutil    字符串工具

github.com/thoas/go-funk    函数式工具库，主要用于遍历数组，



```

可以把其它项目中的util都整理进来
